Voter File Basics for Organizers

This free webinar is designed to teach organizers of small to mid-sized US political campaigns how to use the Python programming language to run a more effective campaign.

No programming background or special software is needed.

Agenda

Understanding who your voters are and how to reach them effectively is key to a successful campaign. In this class, you will learn how to use basic programming in the Python language to gain demographic insights and create strategic voter cohorts from your voter file. This class includes an start to finish scenario for building targeting mailing lists for a school board campaign based on age and past voting participation.

For more details about the skills covered, review the materials here.

Pre-Work

To make the most of class time, after signing up, please do the following before we begin:

  • Open the course materials here - you will need to sign in with a Google/Gmail account and add the Colaboratory add-on.
  • Obtain a copy of your district's voter file - there are a number resources for this, including your local political party, NationBuilder, or directly from your state or county. States may charge an administrative fee to access this information, so check with your party to see what resources are available to you.
